# Usage

```bash

```

## Key input

| Key   | Function               |
| ----- | ---------------------- |
| LEFT  |  Move left              |
| RIGHT |  Move right             |
| DOWN  |  Fall fast              |
| Z     |  Rotate left            |
| X     |  Rotate Right           |
| R     |  Restart game           |
| P     |  Pause and resume game  |
| Q     |  Quit                   |
| W     |  Puyos appear randomly |
